Help & Frequently Asked Questions

1. How to Enter a Valid Path?

  • Provide absolute path (recommended)
  • Example (Windows): C:\Users\YourName\Documents
  • Do NOT include quotes (" ")
  • Ensure folder exists before submitting

2. Structure Search vs System Search

  • Structure Search: Searches only inside the last analyzed folder.
  • Fast and memory-based.
  • System Search: Scans entire system directories.
  • Slower but broader.

3. Session Duration

  • Session remains active for 2 hours.
  • After expiration, login is required again.
  • Inactive sessions are automatically removed.

4. Common Errors & Solutions

  • Invalid Path: Ensure correct spelling and folder exists.
  • Session Expired: Login again.
  • Access Denied: Folder permission issue.
  • Search Not Found: Try analyzing correct folder first.
