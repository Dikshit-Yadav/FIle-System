How to Use System

  1. Login using your registered credentials.
  2. Enter Folder Path in the "Analyze Folder" section.
    Example (Windows): C:\Users\YourName\Documents
  3. Click "Analyze Folder" to generate complete folder structure.
  4. View Results including:
    • Total files count
    • Total folders count
    • Hierarchical directory structure
  5. Search Within Structure to find a specific file or folder.
  6. Follow Navigation Steps provided to locate your file easily.
